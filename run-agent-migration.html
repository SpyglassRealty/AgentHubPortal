🔧 Run Agent Fields Migration

This will add the following fields to agent profiles:

  • years_of_experience - Years in real estate
  • languages - Languages spoken (array)
  • specialties - Areas of expertise (array)
Steps:
  1. Make sure you're logged into Mission Control as an admin
  2. Click the "Run Migration" button below
  3. The migration will add the new columns to the database
  4. Once complete, you can edit agents with the new fields
