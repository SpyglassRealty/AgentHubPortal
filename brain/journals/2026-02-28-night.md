### 2. Agent Onboarding Hub ✅
**Location:** `projects/agent-onboarding/index.html`  
**Commit:** `f3089ef06`

If you're going from 179 to 300 agents, you're onboarding 121+ people. That process needs to be systematic, not ad hoc. This tool ensures every new agent gets the same great experience.

**Features:**
- **5-phase onboarding journey**: Pre-Join → Day One → Week One → First 30 Days → First 90 Days
- **43 tasks** across 6 departments (Admin, Ops, Marketing, Leadership, Training, Recruiting)
- **Priority levels**: Critical, High, Medium — with overdue alerts and critical task warnings
- **Auto-calculated due dates** based on agent's start date
- **Two views**: Checklist (by phase) and By Owner (kanban-style cards per department)
- **Filter by owner or priority** to focus on your responsibilities
- **Per-task notes** field for adding context
- **Markdown export** — generates a downloadable checklist file
- **localStorage persistence** — progress saves between sessions
- **Print-friendly** layout

Zero dependencies — single HTML file, Tailwind CDN. Works offline.
