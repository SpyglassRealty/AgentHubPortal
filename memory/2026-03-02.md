## Saved Search Integration Project Started

### Repliers API Testing Results (live API probing)
- **Agent API**: Created test agent (Ryan, agentId: 106875). Fields: fname, lname, email, phone (11-digit), brokerage, designation
- **Client API**: Created test client (clientId: 163640). Linked to agent, has email/sms/whatsapp preferences
- **Saved Search API**: Successfully created/deleted searches. Key findings:
  - Required: class, type (lowercase "sale"/"lease"), minPrice, maxPrice
  - Location: needs cities+areas, OR neighborhoods, OR map polygon (at least one)
  - boardId: 53 (ACTRIS)
  - 100 match limit on initial results (406 error if too broad)
  - cities/areas/neighborhoods are ARRAYS (not singular like listings search)
  - Supports: beds, baths, sqft, yearBuilt, lotSize, stories, garage, pool, pets, keywords, etc.
  - propertyTypes uses different taxonomy than listings search style param
- **Favorites API**: POST/GET/DELETE working. Returns FULL listing data (no re-fetch needed)
- **Valid values**: areas = Travis/Williamson/Hays, class = residential/condo/commercial
