## QA Test Plan for Recruiting Pipeline App

**Objective:** Thoroughly test all new and existing features of the recruiting pipeline application, focusing on Beacon search enhancements and office search functionalities.

**I. Beacon Search Enhancements**

   1. **Advanced Filters:**
      - Verify that all filters present in the Courted section are now available in the main Beacon search.
      - Test each filter individually and in combination with others to ensure accurate results.
      - Confirm that filters are applied consistently across all search interfaces.

   2. **Office/Brokerage Search:**
      - Test the improved office/brokerage search functionality.
      - Verify that search results accurately reflect the selected office or brokerage.
      - Ensure that the search function handles various office names and addresses correctly.

   3. **Filter Consistency:**
      - Ensure all search interfaces have consistent filter options.
      - Confirm that applying filters in one section of the application reflects accurately in other sections.

**II. Office Search Enhancements**

   1. **Robustness:**
      - Test office-level filtering and search across various scenarios.
      - Ensure search results are accurate and comprehensive.

   2. **Functionality:**
      - Verify that all expected office search capabilities are present and functioning as intended.

**III. Previously Completed Recruiting Pipeline Features**

   1. **Candidate Management:**
      - Verify the creation, editing, and deletion of candidate profiles.

   2. **Workflow Automation:**
      - Test automated email sequences and task assignments.

   3. **Reporting:**
      - Confirm the accuracy of all generated reports.

**IV. General Testing Guidelines**

   - Test on different browsers and devices to ensure compatibility.
   - Pay attention to UI/UX elements for a seamless user experience.
   - Document all test cases, including steps, expected results, and actual results.

**Passing Criteria:**

   - All test cases must pass with expected results.
   - No critical or major bugs should be identified.
