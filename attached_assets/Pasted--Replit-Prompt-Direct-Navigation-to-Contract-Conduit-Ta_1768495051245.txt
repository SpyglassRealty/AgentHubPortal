## Replit Prompt: Direct Navigation to Contract Conduit

---

**Task: Remove Intermediate Page - Direct Link to Contract Conduit**

Currently, when users click "Contract Conduit" in the apps list, they see an intermediate page with an "Open Contract Conduit" button. We want to skip this extra step and directly navigate to the Render-hosted Contract Conduit when clicked.

**Contract Conduit URL:** `https://mission-control-contract-conduit.onrender.com/`

---

### Current Behavior (Remove This)
1. User clicks "Contract Conduit" app card
2. Shows intermediate page: "Contract Conduit opens in a new browser tab..."
3. User clicks "Open Contract Conduit" button
4. Finally opens the app

### Desired Behavior
1. User clicks "Contract Conduit" app card
2. **Directly opens** `https://mission-control-contract-conduit.onrender.com/` in a new tab

---

**Find the app card click handler and update it:**

```typescript

    tags: ['Core', 'Live'],
    openInNewTab: true
  },
  // ... other apps
];

// Simple click handler - direct navigation
const handleAppClick = (app: typeof APPS[0]) => {
  if (app.url) {
    window.open(app.url, '_blank', 'noopener,noreferrer');
  }
};
```

**Update the App Card component:**

```tsx
// AppCard.tsx or within Dashboard.tsx

interface AppCardProps {
  app: {
    id: string;
    name: string;
    description: string;
    url: string;
    icon: React.ReactNode;
    tags: string[];
  };
}

const AppCard = ({ app }: AppCardProps) => {
  const handleClick = () => {
    // Direct navigation - no intermediate page
    window.open(app.url, '_blank', 'noopener,noreferrer');
  };

  return (
    <div className="app-card" onClick={handleClick}>
      <div className="app-card-icon">
        {app.icon}
      </div>
      <h3>{app.name}</h3>
      <p>{app.description}</p>
      <div className="app-tags">
        {app.tags.map(tag => (
          <span key={tag} className={`tag tag-${tag.toLowerCase()}`}>
            {tag}
          </span>
        ))}
      </div>
    </div>
  );
};
```

### Quick Fix - If Using Link/Navigate

If your app cards use React Router's `Link` or `navigate`:

```tsx
// BEFORE (navigates to intermediate page)
<Link to="/app/contract-conduit">
  <AppCard app={contractConduit} />
</Link>

// AFTER (direct external link)
<a 
  href="https://mission-control-contract-conduit.onrender.com/" 
  target="_blank" 
  rel="noopener noreferrer"
  className="app-card-link"
>
  <AppCard app={contractConduit} />
</a>
```

**Or handle in onClick:**

```tsx
<div 
  className="app-card"
  onClick={() => window.open('https://mission-control-contract-conduit.onrender.com/', '_blank')}
>
  {/* card content */}
</div>
```
