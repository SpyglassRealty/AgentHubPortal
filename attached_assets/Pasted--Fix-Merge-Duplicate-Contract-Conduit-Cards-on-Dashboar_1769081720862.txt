### Card 2 (Bottom)
- **Title:** Contract Conduit  
- **Description:** "Turn your marketing leads into contracts. Streamline the transition from prospect to..."

These should be merged into a **single** card.

## Implementation

Find where dashboard cards are defined (likely in a config file or database) and remove the duplicate:

### Option A: If cards are in a config/array

```typescript
// Before - Two separate entries
const dashboardApps = [
  {
    id: 'contract-conduit-1',
    name: 'Contract Conduit',
    description: 'Your first step to entering a transaction.',
    icon: 'document',
    categories: ['Core'],
    status: 'Integrated',
    url: '/contract-conduit'
  },
  // ... other apps ...
  {
    id: 'contract-conduit-2',  // DUPLICATE!
    name: 'Contract Conduit',
    description: 'Turn your marketing leads into contracts...',
    icon: 'document',
    categories: ['Marketing'],
    status: 'Integrated',
    url: '/contract-conduit'
  },
];

// After - Single merged entry
const dashboardApps = [
  {
    id: 'contract-conduit',
    name: 'Contract Conduit',
    description: 'Manage real estate transactions, marketing materials, and CMAs.',
    icon: 'document',
    categories: ['Core', 'Marketing'],  // Combined categories
    status: 'Integrated',
    url: '/contract-conduit'
  },
  // ... other apps ...
];
```

### Option B: If cards are in database

```sql
-- Find duplicates
SELECT * FROM dashboard_apps WHERE name = 'Contract Conduit';

-- Delete the duplicate (keep the one with lower ID or the "Core" one)
DELETE FROM dashboard_apps WHERE id = [duplicate_id];

-- Update the remaining one with merged info
UPDATE dashboard_apps 
SET 
  description = 'Manage real estate transactions, marketing materials, and CMAs.',
  categories = ARRAY['Core', 'Marketing']
WHERE name = 'Contract Conduit' AND id = [keep_id];
```
