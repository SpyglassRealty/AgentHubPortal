### Issue 2: Allow Clicking Past Days

Remove any restriction preventing clicks on past dates:
```typescript
// WRONG - prevents past date clicks
const handleDayClick = (date) => {
  if (date < new Date()) return; // Remove this
  // ...
};

// CORRECT - allow all dates
const handleDayClick = (date) => {
  setSelectedDate(date);
  // Show events for that day
};
```

Users should be able to:
- Click any day (past or future) to see that day's events
- Click any event (past or future) to see details
