Update the favicon and page title for Mission Control.

---

## 1. UPDATE FAVICON

Replace the default Replit favicon with the Spyglass Realty branding.

Steps:

1. Add favicon files to `client/public/` or `public/` folder:
   - Use the orange "S" icon from the Mission Control sidebar for brand consistency
   - Create multiple sizes for browser compatibility:
     * favicon.ico (32x32)
     * favicon-16x16.png
     * favicon-32x32.png
     * apple-touch-icon.png (180x180)

2. Update `index.html` in the <head> section:
```html
<head>
  <link rel="icon" type="image/x-icon" href="/favicon.ico" />
  <link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png" />
  <link rel="icon" type="image/png" sizes="16x16" href="/favicon-16x16.png" />
  <link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png" />
</head>
```

3. If the orange Spyglass icon isn't available as a separate file, extract it from the sidebar component or use `SpyglassRealty_Logo_Black.png` and create an orange version.

---

## 2. UPDATE PAGE TITLE

Change the browser tab title from any default to "Mission Control":

In `index.html`:
```html
<head>
  <title>Mission Control | Spyglass Realty</title>
</head>
```

Or if using React Helmet or similar, update the default title in the app.
