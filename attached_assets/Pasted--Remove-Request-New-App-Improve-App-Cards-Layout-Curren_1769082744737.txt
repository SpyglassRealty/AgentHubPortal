### 5. Alternative: Simple Sorted Grid (No Category Headers)

If category headers feel too heavy, just sort apps logically:
```tsx
// Sort apps by: Integrated first, then Core, then Sales, then Marketing
const sortedApps = [...apps].sort((a, b) => {
  const priorityOrder = {
    'Integrated': 0,
    'Core': 1,
    'Sales': 2,
    'Marketing': 3
  };
  
  // Integrated apps first
  if (a.type === 'integrated' && b.type !== 'integrated') return -1;
  if (b.type === 'integrated' && a.type !== 'integrated') return 1;
  
  // Then by category
  const aPriority = priorityOrder[a.category] ?? 99;
  const bPriority = priorityOrder[b.category] ?? 99;
  
  return aPriority - bPriority;
});

// Render
<div className="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 xl:grid-cols-4 gap-4">
  {sortedApps.map((app) => (
    <AppCard key={app.id} app={app} />
  ))}
</div>
```
