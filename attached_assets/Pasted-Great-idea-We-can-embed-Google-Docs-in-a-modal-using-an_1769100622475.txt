### Add Google Docs Modal Viewer for Handbook & Brand Guidelines

**Goal:**
Instead of opening Google Docs in a new tab, display them in a modal within Mission Control. The modal should respect light/dark/system theme.

**Documents to embed:**
1. **Company Handbook** - `https://docs.google.com/document/d/13EJYTRJ9QAXOdJH1H6Vyvh6iBaUsVJcUORxw6PuDcVM/edit`
2. **Brand Guidelines** - `https://docs.google.com/document/d/1LKeF3DPqWelqF-ESWzJe3FEVewVTNaDCt4yv3rMz740/edit`

**Note:** Google Docs iframe embeds always have a white background (Google limitation). We'll style the modal container to match the theme, but the document itself will remain light.

---

## 1. Create Google Doc Modal Component
```tsx
// components/GoogleDocModal.tsx

import { useState } from 'react';
import { X, ExternalLink, Maximize2, Minimize2, Loader2 } from 'lucide-react';
import { useTheme } from '@/contexts/ThemeContext';

interface GoogleDocModalProps {
  isOpen: boolean;
  onClose: () => void;
  title: string;
  docId: string;
  externalUrl?: string;
}

export function GoogleDocModal({ 
  isOpen, 
  onClose, 
  title, 
  docId,
  externalUrl 
}: GoogleDocModalProps) {
  const { theme } = useTheme();
  const isDark = theme === 'dark';
  const [isFullscreen, setIsFullscreen] = useState(false);
  const [isLoading, setIsLoading] = useState(true);

  if (!isOpen) return null;

  // Google Docs embed URL (preview mode for read-only viewing)
  const embedUrl = `https://docs.google.com/document/d/${docId}/preview`;
  
  // Theme classes
  const modalBg = isDark ? 'bg-gray-900' : 'bg-white';
  const headerBg = isDark ? 'bg-gray-800' : 'bg-gray-100';
  const textPrimary = isDark ? 'text-white' : 'text-gray-900';
  const textSecondary = isDark ? 'text-gray-400' : 'text-gray-600';
  const borderColor = isDark ? 'border-gray-700' : 'border-gray-200';
  const overlayBg = isDark ? 'bg-black/80' : 'bg-black/50';

  return (
    <div 
      className={`fixed inset-0 z-50 flex items-center justify-center p-4 ${overlayBg}`}
      onClick={onClose}
    >
      <div 
        className={`
          relative flex flex-col
          ${isFullscreen ? 'w-full h-full' : 'w-full max-w-5xl h-[90vh]'}
          ${modalBg} rounded-xl shadow-2xl overflow-hidden
          transition-all duration-300
        `}
        onClick={(e) => e.stopPropagation()}
      >
        {/* Header */}
        <div className={`flex items-center justify-between px-4 py-3 ${headerBg} border-b ${borderColor}`}>
          <div className="flex items-center gap-3">
            {/* Google Docs Icon */}
            <div className="w-8 h-8 bg-blue-500 rounded flex items-center justify-center">
              <svg className="w-5 h-5 text-white" viewBox="0 0 24 24" fill="currentColor">
                <path d="M14 2H6a2 2 0 0 0-2 2v16a2 2 0 0 0 2 2h12a2 2 0 0 0 2-2V8l-6-6zm-1 2l5 5h-5V4zM6 20V4h6v6h6v10H6z"/>
                <path d="M8 12h8v2H8zm0 4h8v2H8z"/>
              </svg>
            </div>
            <div>
              <h2 className={`font-semibold ${textPrimary}`}>{title}</h2>
              <p className={`text-xs ${textSecondary}`}>Google Document</p>
            </div>
          </div>
          
          <div className="flex items-center gap-2">
            {/* Open in new tab */}
            {externalUrl && (
              
                href={externalUrl}
                target="_blank"
                rel="noopener noreferrer"
                className={`p-2 rounded-lg hover:bg-gray-200 dark:hover:bg-gray-700 transition-colors ${textSecondary}`}
                title="Open in new tab"
              >
                <ExternalLink className="w-5 h-5" />
              </a>
            )}
            
            {/* Fullscreen toggle */}
            <button
              onClick={() => setIsFullscreen(!isFullscreen)}
              className={`p-2 rounded-lg hover:bg-gray-200 dark:hover:bg-gray-700 transition-colors ${textSecondary}`}
              title={isFullscreen ? 'Exit fullscreen' : 'Fullscreen'}
            >
              {isFullscreen ? (
                <Minimize2 className="w-5 h-5" />
              ) : (
                <Maximize2 className="w-5 h-5" />
              )}
            </button>
            
            {/* Close button */}
            <button
              onClick={onClose}
              className={`p-2 rounded-lg hover:bg-gray-200 dark:hover:bg-gray-700 transition-colors ${textSecondary}`}
              title="Close"
            >
              <X className="w-5 h-5" />
            </button>
          </div>
        </div>

        {/* Document Content */}
        <div className="flex-1 relative bg-gray-100 dark:bg-gray-800">
          {/* Loading indicator */}
          {isLoading && (
            <div className="absolute inset-0 flex items-center justify-center bg-gray-100 dark:bg-gray-800 z-10">
              <div className="flex flex-col items-center gap-3">
                <Loader2 className={`w-8 h-8 animate-spin ${textSecondary}`} />
                <p className={textSecondary}>Loading document...</p>
              </div>
            </div>
          )}
          
          {/* Google Docs iframe */}
          <iframe
            src={embedUrl}
            className="w-full h-full border-0"
            onLoad={() => setIsLoading(false)}
            title={title}
            allow="autoplay"
          />
        </div>

        {/* Footer - Optional info bar */}
        <div className={`px-4 py-2 ${headerBg} border-t ${borderColor} flex items-center justify-between`}>
          <p className={`text-xs ${textSecondary}`}>
            ğŸ’¡ Tip: Use Ctrl/Cmd + F to search within the document
          </p>
          <p className={`text-xs ${textSecondary}`}>
            Viewing in read-only mode
          </p>
        </div>
      </div>
    </div>
  );
}
```

## 6. Important Notes

**Google Docs Limitation:**
- Google Docs embeds always have a white background
- We cannot apply dark mode to the document content itself
- The modal container, header, and footer will match the theme
- In dark mode, we add a subtle dark gray padding around the iframe to soften the contrast

**Embed URL Format:**
- Use `/preview` for read-only viewing: `https://docs.google.com/document/d/{DOC_ID}/preview`
- This removes the Google Docs UI (toolbar, menus) for a cleaner look

**Permissions:**
- Documents must be set to "Anyone with link can view" for embedding to work
- If documents are private, users will see a "Request access" screen
