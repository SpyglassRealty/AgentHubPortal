Add Tooltips to My Performance Cards
Overview
Add informational tooltips (hover or click info icon) to each metric card on the My Performance page. Tooltips should explain what the metric means and where the data comes from.
Implementation
Add a small info icon (ℹ️ or ?) to the top-right corner of each card. On hover (desktop) or tap (mobile), show a tooltip with the explanation.

Component Implementation
typescript// components/InfoTooltip.tsx
import { Tooltip } from '@/components/ui/tooltip'; // or your UI library
import { Info } from 'lucide-react'; // or similar icon

interface InfoTooltipProps {
  content: string;
  source?: string;
}

export function InfoTooltip({ content, source }: InfoTooltipProps) {
  return (
    <Tooltip>
      <TooltipTrigger asChild>
        <button className="text-gray-400 hover:text-gray-600 ml-1">
          <Info className="h-4 w-4" />
        </button>
      </TooltipTrigger>
      <TooltipContent className="max-w-xs p-3">
        <p className="text-sm">{content}</p>
        {source && (
          <p className="text-xs text-gray-500 mt-1">Source: {source}</p>
        )}
      </TooltipContent>
    </Tooltip>
  );
}

Success Criteria

 Each metric card has a small info icon (ℹ️)
 Hovering/tapping shows a tooltip with explanation
 Tooltips include what the metric means
 Tooltips include the data source (ReZen)
 Works on both desktop (hover) and mobile (tap)
 Tooltips don't block important content
 Consistent styling across all cards
