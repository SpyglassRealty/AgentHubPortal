### Fix: Training Section Dark Mode - White Background Issue

**Issue:** The Training section has large white/light areas in dark mode:
- Main page background is white instead of dark
- Video cards have white backgrounds
- Overall doesn't match the dark sidebar

**Solution:** Ensure all backgrounds properly sync with theme.

---

## 1. Fix Page Background

Find the main container in `training.tsx` and ensure it has the correct background:
```tsx
// The main page wrapper MUST have the theme background
return (
  <div className={`min-h-screen ${isDark ? 'bg-gray-900' : 'bg-gray-50'} transition-colors duration-200`}>
```

**IMPORTANT:** Make sure there's no parent wrapper overriding this with a white background.

---

## 2. Check for Conflicting Parent Styles

The issue might be a parent layout component. Check if Training page is wrapped in a layout that has a white background.

Look in your layout/App structure for something like:
```tsx
// BAD - Parent has white background
<div className="bg-white">
  <TrainingPage />
</div>

// GOOD - Parent respects theme OR is transparent
<div className={isDark ? 'bg-gray-900' : 'bg-white'}>
  <TrainingPage />
</div>

// OR just remove background from parent
<div>
  <TrainingPage />
</div>
```

---

## 3. Fix Main Content Area

If there's a main content wrapper, update it:
```tsx
// Find the main content area wrapper (often in a Layout component)
// It might look like this:

// BEFORE (forcing white):
<main className="flex-1 overflow-auto bg-white">

// AFTER (theme-aware):
<main className={`flex-1 overflow-auto ${isDark ? 'bg-gray-900' : 'bg-gray-50'}`}>
```

## 4. Complete Training Page Dark Mode Fix

Replace the theme classes at the top of the TrainingPage component:
```tsx
export default function TrainingPage() {
  const { theme } = useTheme();
  const isDark = theme === 'dark';
  
  // ============================================
  // THEME CLASSES - COMPLETE DARK MODE SUPPORT
  // ============================================
  
  // Backgrounds
  const pageBg = isDark ? 'bg-gray-900' : 'bg-gray-50';
  const cardBg = isDark ? 'bg-gray-800' : 'bg-white';
  const headerBg = isDark ? 'from-gray-900/95 via-gray-900/80' : 'from-gray-50/95 via-gray-50/80';
  const inputBg = isDark ? 'bg-gray-800' : 'bg-white';
  const modalBg = isDark ? 'bg-gray-900' : 'bg-white';
  const emptyStateBg = isDark ? 'bg-gray-800/50' : 'bg-gray-100';
  
  // Text
  const textPrimary = isDark ? 'text-white' : 'text-gray-900';
  const textSecondary = isDark ? 'text-gray-300' : 'text-gray-600';
  const textMuted = isDark ? 'text-gray-400' : 'text-gray-500';
  
  // Borders
  const borderColor = isDark ? 'border-gray-700' : 'border-gray-200';
  const inputBorder = isDark ? 'border-gray-700' : 'border-gray-300';
  
  // Interactive
  const hoverBg = isDark ? 'hover:bg-gray-700' : 'hover:bg-gray-100';
  const buttonSecondary = isDark 
    ? 'bg-gray-800 hover:bg-gray-700 text-white border-gray-700' 
    : 'bg-white hover:bg-gray-100 text-gray-900 border-gray-300';

  // ... rest of component
```
