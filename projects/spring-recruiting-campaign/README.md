# Spring 2026 Recruiting Campaign Kit

Complete outreach toolkit for Spyglass Realty's push to 300 agents.

## What's Inside

1. **Email Templates** (5 sequences) — Cold outreach, follow-up, post-meeting, referral, re-engage
2. **Talking Points** — Data-driven scripts for phone/in-person conversations
3. **Social Media Content** — LinkedIn/Instagram posts highlighting Spyglass value prop
4. **Recruiting One-Pager** — Print-ready summary for meetings
5. **Objection Handlers** — Top 15 objections with data-backed responses

## How to Use

- Email templates have `{{MERGE_FIELDS}}` — fill with agent data from the Command Center
- Talking points reference the Commission Calculator and Production Planner
- All stats sourced from MLS Grid data (ACTRIS) and market research
