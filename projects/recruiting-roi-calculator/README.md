# Spyglass Realty — Agent ROI Calculator

A recruiting tool that lets prospective agents see how much more they could earn at Spyglass Realty.

## What It Does
- Agents input their current sales volume, transactions, commission rate, and brokerage terms
- Pre-built presets for KW, Compass, eXp, Realty Austin, REAL
- Side-by-side comparison with Spyglass Realty terms
- Shows annual and per-transaction difference
- Visual comparison bars
- CTA to schedule a conversation

## Tech
- Single HTML file (zero dependencies, no build step)
- Dark theme matching Spyglass brand
- Fully responsive
- Can be deployed anywhere: Vercel, Netlify, embedded in spyglassrealty.com, or sent as a link

## Deployment Options
1. **Standalone page**: Deploy to Vercel/Netlify
2. **Embed**: Add to spyglassrealty.com recruiting page
3. **Share link**: Host on a subdomain like `calc.spyglassrealty.com`

## Customization
- Update Spyglass split/fees/cap defaults to match current offerings
- Preset brokerage data can be updated as market changes
- CTA link points to agent FAQ page — update as needed
